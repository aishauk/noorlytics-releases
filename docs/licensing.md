# Licensing & Usage

Noorlytics uses a usage-based licensing model.

---

## What is counted?

Each analysis run counts toward your usage.

Examples:

- running `noor analyze`
- running `noor suggest`
- running `noor add-tests`

---

## What is not counted?

- checking license status
- local configuration changes

---

## What happens when I reach the limit?

You will not be able to run new analyses until your usage resets or your plan is updated.

---

## Device usage

Licenses may be tied to a device to prevent abuse.

---

## Why is usage tracked?

We only track usage metrics (like number of runs).

Your code is never collected or stored when using local mode.
