# Security & Privacy

Noorlytics is designed with a local-first approach.

---

## Local mode (default)

When using Ollama:

- your code stays on your machine
- no files are uploaded
- no external processing is performed

This is the recommended mode.

---

## Cloud mode (optional)

When using OpenAI or external compute:

- parts of your code may be sent to external services
- processing happens outside your environment

Only use this if you are comfortable with that trade-off.

---

## What Noorlytics tracks

We only track usage metrics:

- number of runs
- license validation

We do not collect or store your code.

---

## Summary

- Local mode = full control  
- Cloud mode = optional convenience  

Noorlytics is built to keep your code where it belongs: on your machine.
